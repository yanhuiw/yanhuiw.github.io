Java 程序流程控制

Java 程序有 3 种程序执行的流程：

  • 顺序执行：默认的执行流程
  • 条件判断：if-else，switch-case
  • 循环执行：for，while，do-while

条件判断

if-else

switch-case

相当于 if-else 串联。

使用 break 退出 switch 流程。

穿透效应：如果没有break，程序会继续执行下一个 case 的功能。

从 Java 12 开始，使用类似模式匹配的方法，保证只有一个 case 被执行，不需要 break 语句。如果一个case包含多条语句，需要使用 {} 将多条语句包括进来。如果需要复杂语句，可以使用yield 返回值。

循环执行

让计算机根据循环条件做循环计算，条件满足时继续循环，条件不满足时推出循环。

也可以使用 break 和 continue 结束循环：

  • break：跳出循环。

  • continue：提前结束本次循环操作，进入下一次循环。

for

包含初始条件，循环条件，更新语句。可以省略任意一部分。

while

先判断循环条件，再进行循环操作。

do-while

先进行一次循环操作，再判断循环条件。
