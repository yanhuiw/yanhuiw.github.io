Posted on 2019-10-29 In Java Views:

Java 程序组成

一个 Java 程序可以认为是一系列对象的集合，这些对象通过调用彼此的方法来协同工作。

  • 对象：对象是类的一个实例，有状态和行为。
  • 类：类是一个模板，描述一类对象的状态和行为。
  • 方法：对象的行为描述。一个类可以有多个方法。
  • 实例变量：对象的状态描述。每个对象都有独特的实例变量。

Java 基本语法

  • 类名：类的首字母大写。如果类名由若干单词组成，每个单词的首字母大写。例：MyFirstJavaClass。

  • 方法名：方法名的首字母小写。如果方法名由若干单词组成，从第二个单词开始，每个单词的首字母大写。例：getName。

  • 主方法入口：要求入口为静态方法，方法名为 main，参数为 String 数组。即所有的 Java 程序都由 public static void main(String args[])方法开始执行。

  • 源文件名：源文件名必须和类名相同，文件名的后缀为 .java。例：Hello.java。

Java 命名规则

Java 所有的组成部分都需要名字。类名，变量名，方法名等都被称为标识符。

命名规则如下：

  • 所有名称都以字母 （A-Z或者a-z），美元符（$）或下划线（_）开始。首字符之后的字符没有限制，可以是任何字符的组合。

  • 大小写敏感。

  • 关键字不能用作标识符。

Java 注释

注释用于阅读，不是程序的一部分，编译器会自动忽略注释。

Java 有 3 种注释，分别如下：

  • 单行注释：以双斜线//开头，直到这一样的结尾结束。

  • 多行注释：以斜线星号/*开头，以星号斜线*/结束。

  • 自动创建文档注释：以斜线双星号/**开头，以星号斜线*/结束。如果有多行，每行以星号*开头。通常写在类和方法的定义处，用于自动创建文档。
