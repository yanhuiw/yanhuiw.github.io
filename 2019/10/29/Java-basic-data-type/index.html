Java 基本数据类型

Posted on 2019-10-29 In Java Views:

Java 基本数据类型包括：

  • 内置数据类型
  • 引用数据类型

内置数据类型

Java 提供了 8 种基本数据类型。

  • 6 种数字类型
    • 4 种整型：byte, short, int, long
    • 2 种浮点型：float, double
  • 1 种字符类型：char
  • 1 种布尔型：boolean

整型

  • byte：8位，有符号，范围 -128-127，默认值为0
  • short：16位，有符号，范围 -32768-32767，默认值为0
  • int：32位，有符号，范围 -2147483648-2147483647，默认值为0
  • long：64位，有符号，范围 -9223372036854775808-9223372036854775807，默认值为0L

浮点型

  • float：32位，单精度，默认值为0.0f
  • double：64位，双精度，默认值为0.0d

字符型

  • char：16位，Unicode字符，范围 \u0000-\uffff

布尔型

  • boolean：1位，默认值为 false

引用数据类型

  • 引用类型变量由类的构造函数创建，可以使用它们访问所引用的对象。这些变量在声明时被指定为一个特定的类型，声明后类型不能改变。
  • 对象，数组，字符串等都是引用数据类型。
  • 引用类型的默认值为 null。
  • 引用变量可以引用任何与之兼容的类型。
