Java 简介

Java 是由 Sun Microsystems 公司于1995年5月推出的 Java 面向对象程序设计语言和 Java 平台的总称。由 James Gosling 和同事们共同研发，并在1995年正式推出。

Java 语言

Java 编程语言是一个简单，面向对象，分布式，解释性，健壮，安全与系统无关，可移植，高性能，多线程和动态的语言。

Java 不同于一般的编译语言或解释型语言。它首先将源码编译成字节码，再在不同平台上的虚拟机来执行字节码，具有 ”一次编写，到处运行“ 的跨平台特性。

Java 平台

Java 平台分为三个体系：

  • JavaSE (J2SE) (Java2 Platform Standard Edition，java 平台标准版）
  • JavaEE (J2EE) (Java 2 Platform,Enterprise Edition，java 平台企业版)
  • JavaME (J2ME) (Java 2 Platform Micro Edition，java 平台微型版)

JVM

Java 虚拟机（Java Virtual Machine）。JVM 是一个虚构的计算机，有自己完善的硬件架构和指令系统。JVM 通过在实际的计算机上仿真模拟各种计算机的功能来实现。

JVM 的本质是一个程序，当它在命令行上启动的时候，就开始执行保存在某字节码文件（.class）中的指令。任何平台只要装有针对该平台的 JVM，由 Java 源码编译而成的字节码文件就可以在该平台上运行。

JVM 屏蔽了平台相关的信息，使得 Java 源码只需要生成字节码，就可以在不同的平台不加修改地运行，不需要重新编译，实现 ”一次编写，到处运行”。

JRE

Java 运行环境（Java Runtime Environment），包括 JVM 和一些标准类库（ Class Library ）。Java 字节码可以在 JRE 上运行。

JDK

Java 开发工具包（Java Development Kit），是使用最广泛的 Java SDK。包括 JRE，编译器，调试器等一系列的开发工具。
